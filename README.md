# 📚 Library Management System (C++)

A console-based Library Management System built using C++. This project allows users to add books, view all book records, and search for books using their ID. It demonstrates the implementation of Object-Oriented Programming (OOP) and File Handling concepts in C++.

---

## 🚀 Features

* 📖 Add New Books
* 🔍 Search Books by ID
* 📚 View All Books
* 💾 File-Based Data Storage
* 🖥️ Menu-Driven Interface
* 📂 Persistent Record Management

---

## 🛠️ Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling (`fstream`)
* Standard Library

## ⚙️ How It Works

### ➕ Add Book

Users can add a new book by entering:

* Book ID
* Book Name
* Author Name

### 📚 Show Books

Displays all books stored in the library database.

### 🔍 Search Book

Users can search for a specific book using its Book ID.

### 💾 Data Storage

All records are stored in a text file (`libData.txt`) and remain available after the application is closed.
